# Gold-Standard Benchmarks

This folder stores concrete benchmark images for recurring `visual-prompt-router`
deliverable families.

It exists so the skill can say more than `use case-first`.

The intended order is:

1. project-local approved image
2. this benchmark folder
3. weaker prompt-only precedent

## Image Classes

- `direct public benchmark`
  - copied locally from a public source because the single image already anchors
    the family well
- `assembled benchmark board`
  - composed locally from several public clues when one exact public image could
    not be found
- `project-local approved benchmark`
  - copied from a local project because it already matches the subtype better
    than public references
